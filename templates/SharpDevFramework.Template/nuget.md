# SharpDevFramework.Template

SharpDevFramework 项目模板 - 基于 .NET 10.0 的全栈 Web 开发框架。

## 安装

```bash
dotnet new install SharpDevFramework.Template
```

## 使用

```bash
dotnet new sharpdev -n MyProject
```

### 参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-n` | 项目名称 | - |
| `--serviceName` | Windows 服务名称 | `myapp` |
